# Tienda Online - Postura Canalla

Sitio estático inicial (HTML/CSS/JS) para una web de afiliación de Amazon enfocada en ergonomía y productividad para teletrabajo.

## Ejecutar en local
Abre `index.html` en el navegador o usa un servidor estático:

```bash
python3 -m http.server 8000
```

Después visita `http://localhost:8000`.
